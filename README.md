# AndreyGrin Home Assistant Add-ons

This repository contains custom Home Assistant add-ons.

## Available add-ons

- `ev_charge_controller` - Laravel-based EV charging planner and controller

## Installation

Add this repository URL to Home Assistant:

```text
https://github.com/AndreyGrin/ha-charger-control
```

Then install the `EV Charge Controller` add-on from the add-on store.
